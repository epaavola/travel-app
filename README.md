# Adventure tours
Full Stack web-application example including public website for tours, admin dashboard and java backend.

Demo site: https://travel.paavolagroup.fi

### Technologies used:
- Java
- Spring boot
- JavaScript
- React.js
- MySQL
- Keycloak
- Material-ui
- Docker
- Jenkins
